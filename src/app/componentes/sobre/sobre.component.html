FUNCIONALIDADES DO PROJETO

  • Ao clicar no botão "Consumir API" você será redirecionado a uma listagem com os dados dos estados brasileiros que contraíram Coronavírus (COVID-19). Esta listagem foi retirada através do consumo de uma API localizada em: Covid 19 - Brasil
  • Ao clicar no botão "Drag & Drop" você será direcionado a uma listagem de cursos, as quais você poderá ordenar (arrastando o mouse dentro de um contâiner pré-determinado). As propriedades utilizadas foram: cdkDragBoundary para manter o item da lista dentro do contâiner, cdkDropList e cdkDropListDropped para apagar e movimentar o item para nova posição, respectivamente.
  • Ao clicar no botão "Reactive Forms", irá aparecer para você um formulário simples. Quando clicar no botão "ENVIAR" será exibido um texto (formato JSON) com as informações anteriormente digitadas no formulário. Em seu desenvolvimento foram utilizadas as seguintes propriedades: FormGroup , para agrupar o formulário e a propriedade FormControl , que relaciona os campos do formulário com as informações digitadas pelo usuário.
